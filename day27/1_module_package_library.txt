1. Each python file in a python project is a module 
2. Collection of modules is called as Python Package. 
3. Collection of such packages is called Python Library .

Diffrent ways to import in Python
1. import <built-in_package>/<installed_packages>
    e.gg. import json
2. from <package> import <module>
3. from <package>.<module> import <function>/<class>
4. from <module> import <function>/<class>

5. We can also give a diffrent name to the packages while importing 
    e.g. import pandas as pd

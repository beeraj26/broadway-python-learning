Initialize A repo In Local. 
1. git init  (But make sure you are in correct project location)

2.  git status (Check the correct status of the repo.)

3. git add. (Add all the changes to git repo)

4. git branch 

5. git remote -v (List all the git branches)

4. git commit -m "<commit_message>" (List all the remote urls)
